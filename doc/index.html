Name

xm2nes — converts Fasttracker ][ eXtended Module (XM) files to Kent's NES music format

Synopsis

xm2nes [ option ...] {file}

Description

xm2nes reads an eXtended Module (XM) file and converts it to the format used by Kent's NES music player.

Options

--output=file

Store the output in file. If this option is not specified, the output will be written to standard output.

--verbose

Print progress information to standard output.

--help

Give a help list.

--usage

Give a short usage message.

--version

Print program version.

Files

Output

The output is assembly source code.

Examples

To convert mysong.xm:

xm2nes mysong.xm
