Name

xm2nes — converts Fasttracker ][ eXtended Module (XM) files to Kent's NES music format

Synopsis

xm2nes [ option ...] {file}

Description

xm2nes reads an eXtended Module (XM) file and converts it to the format used by Kent's NES music player.

Options

--output=file

Store the output in file. If this option is not specified, the output will be written to standard output.

--channels=channels

Only process the channels defined by the list of channels.

--instruments-map=file

Reads instrument mapping information from file.

--label-prefix=prefix

Prefix 6502 assembly labels by prefix.

--order-start=offset

Start reading pattern order table from offset.

--order-end=offset

Stop reading pattern order table at offset.

--verbose

Print progress information to standard output.

--help

Give a help list.

--usage

Give a short usage message.

--version

Print program version.

Files

Instrument Mapping File

This input file is optional; it enables custom mapping of input instruments to output instruments. The file is a plaintext file, each line containing one or more name:value pairs. Example:

source:10 target:0 transpose:-6

The above line tells the converter to map instrument 10 in the input to instrument 0 in the output, and to transpose each note for that instrument by -6 in the output.

Output

The output is assembly source code.

Examples

To convert mysong.xm:

xm2nes mysong.xm

To convert only channels 0 and 2 of mysong.xm:

xm2nes --channels=0,2 mysong.xm

To convert only the order table range [3,5] of mysong.xm, and call the result "middle":

xm2nes --order-start=3 --order-end=5 --label-prefix=middle mysong.xm
